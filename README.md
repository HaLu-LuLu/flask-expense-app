# 家計簿アプリ
シンプルな家計管理アプリです。
支出の追加・編集・削除、月ごとの表示、カテゴリ別グラフ、検索機能を実装してます。

## 公開URL
https://flask-expense-app.onrender.com

## 主な機能
‐ 支出の追加
‐ 編集・削除
‐ 月ごとの表示
‐ カテゴリ別グラフ表示
‐ メモ検索

## 使用技術
‐ Python(Flask)
‐ HTML / CSS
‐ JavaScript (Chart.js)
‐ Render
‐ Git / GitHub
